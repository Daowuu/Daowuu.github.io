Cut and Stick

发表于 2021-04-21 分类于 数据结构
本文字数： 3.4k 阅读时长 ≈ 3 分钟
在一个静态数组上，每次选取一个区间，该区间可以分为多个区间（保证连续或可以不分），但要保证每个区间的众数不超过区间长度的一半，问最少可以分成几个区间？

Cut and Stick

Baby Ehab has a piece of Cut and Stick with an array aaa of length nnn written on it. He plans to grab a pair of scissors and do the following to it:

  • pick a range (l,r)(l,r)(l,r) and cut out every element al,al+1,...,ara_l, a_{l+1}, ..., a_ral​,al+1​,...,ar​ in this range;
  • stick some of the elements together in the same order they were in the array;
  • end up with multiple pieces, where every piece contains some of the elements and every element belongs to some piece.

More formally, he partitions the sequence al,al+1,...,ara_l, a_{l+1}, ..., a_ral​,al+1​,...,ar​ into subsequences. He thinks a partitioning is beautiful if for every piece (subsequence) it holds that, if it has length xxx, then no value occurs strictly more than ⌈x2⌉⌈\frac x 2⌉⌈2x​⌉ times in it.

He didn’t pick a range yet, so he’s wondering: for qqq ranges (l,r)(l,r)(l,r), what is the minimum number of pieces he needs to partition the elements al,al+1,...,ara_l, a_{l+1}, ..., a_ral​,al+1​,...,ar​ into so that the partitioning is beautiful.

A sequence bbb is a subsequence of an array aaa if bb can be obtained from aaa by deleting some (possibly zero) elements. Note that it does not have to be contiguous.

Input

The first line contains two integers nnn and qqq (1≤n,q≤3⋅105)(1≤n,q≤3⋅10^5)(1≤n,q≤3⋅105) — the length of the array aa and the number of queries.

The second line contains nnn integers a1,a2,...,an(1≤ai≤n)a_1, a_2, ..., a_n (1≤ai≤n)a1​,a2​,...,an​(1≤ai≤n) — the elements of the array aaa.

Each of the next qq lines contains two integers lll and rrr (1≤l≤r≤n)(1≤l≤r≤n)(1≤l≤r≤n) — the range of this query.

Output

For each query, print the minimum number of subsequences you need to partition this range into so that the partitioning is beautiful. We can prove such partitioning always exists.

Note
In the first query, you can just put the whole array in one subsequence, since its length is 6, and no value occurs more than 3 times in it.

In the second query, the elements of the query range are [3,2,3,3]. You can’t put them all in one subsequence, since its length is 4, and 3 occurs more than 2 times. However, you can partition it into two subsequences: [3] and [2,3,3].

思路

用可持久化线段树（主席树）判断区间是否需要分多组，若不需要返回 −1-1−1，若需要返回众数个数，判断的标准即为众数个数是否大于区间的 ⌈x2⌉⌈\frac x 2⌉⌈2x​⌉ 。

若判断出区间要分多组，把众数贡献看作 111，非众数贡献看作 −1-1−1，做前缀和，那么分出的第 iii 个区间的结尾是最后一个前缀和为 iii 的点，

例如：5 5 1 5 5 1 5 5 2 3， 前缀和为 1 2 1 2 3 2 3 4 3 2，所以第一个区间是 [1,3][1,3][1,3]，第二个区间是 [4,10][4,10][4,10]。

可以明显看出区间的分组个数就是前 nnn 项前缀和，即 最小分组数 = 众数个数 - 非众数个数​。

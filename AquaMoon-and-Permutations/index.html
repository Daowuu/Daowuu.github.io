AquaMoon and Permutations

time limit per test: 2 seconds

memory limit per test: 256 megabytes

Articles

Cirno has prepared nnn arrays of length nnn each. Each array is a permutation of nnn integers from 111 to nnn. These arrays are special: for all 1≤i≤n1≤i≤n1≤i≤n, if we take the iii-th element of each array and form another array of length nnn with these elements, the resultant array is also a permutation of nnn integers from 111 to nnn. In the other words, if you put these nnn arrays under each other to form a matrix with nnn rows and nn columns, this matrix is a Latin square.

Afterwards, Cirno added additional nnn arrays, each array is a permutation of nnn integers from 11 to nnn. For all 1≤i≤n1≤i≤n1≤i≤n, there exists at least one position 1≤k≤n1≤k≤n1≤k≤n, such that for the iii-th array and the (n+i)(n+i)(n+i)-th array, the kkk-th element of both arrays is the same. Notice that the arrays indexed from n+1n+1n+1 to 2n2n2n don’t have to form a Latin square.

Also, Cirno made sure that for all 2n2n2n arrays, no two arrays are completely equal, i. e. for all pair of indices 1≤i<j≤2n1≤i<j≤2n1≤i<j≤2n, there exists at least one position 1≤k≤n1≤k≤n1≤k≤n, such that the kkk-th elements of the iii-th and jjj-th array are different.

Finally, Cirno arbitrarily changed the order of 2n2n2n arrays.

AquaMoon calls a subset of all 2n2n2n arrays of size nnn good if these arrays from a Latin square.

AquaMoon wants to know how many good subsets exist. Because this number may be particularly large, find it modulo 998244353998244353998244353. Also, she wants to find any good subset. Can you help her?

Input

The input consists of multiple test cases. The first line contains a single integer t(1≤t≤100)t (1≤t≤100)t(1≤t≤100) — the number of test cases.

The first line of each test case contains a single integer n(5≤n≤500)n (5≤n≤500)n(5≤n≤500).

Then 2n2n2n lines followed. The iii-th of these lines contains nnn integers, representing the iii-th array.

It is guaranteed, that the sum of nnn over all test cases does not exceed 500500500.

Output

For each test case print two lines.

In the first line, print the number of good subsets by modulo 998244353998244353998244353.

In the second line, print nnn indices from 111 to 2n2n2n — indices of the nnn arrays that form a good subset (you can print them in any order). If there are several possible answers — print any of them.

Tutorial

Among all the arrays not be chosen, if an array have a number which appears exactly once at its column, that the array must belong to the n original arrays. So, we can choose the array and delete all arrays have at least one same bit with it.

If there not exists such an array discribed above, according to the Pigeonhole Principle, all numbers of the unchosen arrays must appear exactly twice on their columns. It means either the original arrays or the additional arrays can form a correct latin square. So, it’s correct to choose any one of the unchosen array and delete all arrays have at least one same bit with it. Meanwhile, we need to multiply the number of solutions by 2.

We need to repeat the above operations, until we have chosen n arrays.

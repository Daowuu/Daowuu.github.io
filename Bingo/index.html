memory limit per test: 512 megabytes

Articles

Getting ready for VK Fest 2021, you prepared a table with nnn rows and nn columns, and filled each cell of this table with some event related with the festival that could either happen or not: for example, whether you will win a prize on the festival, or whether it will rain.

Forecasting algorithms used in VK have already estimated the probability for each event to happen. Event in row ii and column jj will happen with probability ai,j⋅10−4a_{i,j}⋅10^{−4}ai,j​⋅10−4. All of the events are mutually independent.

Let’s call the table winning if there exists a line such that all nnn events on it happen. The line could be any horizontal line (cells (i,1),(i,2),…,(i,n)(i,1),(i,2),…,(i,n)(i,1),(i,2),…,(i,n) for some ii), any vertical line (cells (1,j),(2,j),…,(n,j)(1,j),(2,j),…,(n,j)(1,j),(2,j),…,(n,j) for some jjj), the main diagonal (cells (1,1),(2,2),…,(n,n)(1,1),(2,2),…,(n,n)(1,1),(2,2),…,(n,n)), or the antidiagonal (cells (1,n),(2,n−1),…,(n,1)(1,n),(2,n−1),…,(n,1)(1,n),(2,n−1),…,(n,1)).

Find the probability of your table to be winning, and output it modulo 316073160731607 (see Output section).

Input

The first line contains a single integer n(2≤n≤21)n (2≤n≤21)n(2≤n≤21) — the dimensions of the table.

The iii-th of the next nnn lines contains nnn integers ai,1,ai,2,…,ai,nai,1,ai,2,…,ai,nai,1,ai,2,…,ai,n (0<ai,j<1040<a_{i,j}<10^40<ai,j​<104). The probability of event in cell (i,j)(i,j)(i,j) to happen is ai,j⋅10−4a_{i,j}⋅10^{−4}ai,j​⋅10−4.

Output

Print the probability that your table will be winning, modulo 316073160731607.

Formally, let M=31607M=31607M=31607. It can be shown that the answer can be expressed as an irreducible fraction pq\frac{p}{q}qp​, where ppp and qqq are integers and q≢0(modM). Output the integer equal to p⋅q−1mod Mp⋅q^{−1} mod \ Mp⋅q−1mod M. In other words, output such an integer xxx that 0≤x<M0≤x<M0≤x<M and x⋅q≡p(mod M)x⋅q≡p(mod\ M)x⋅q≡p(mod M).

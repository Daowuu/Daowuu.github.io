Product 1 Modulo N

Now you get Baby Ehab’s first words: “Given an integer nnn, find the longest subsequence of 1,2,…,n−11,2,…,n−11,2,…,n−1 whose product is 111 modulo nnn.” Please solve the problem.

A sequence bbb is a subsequence of an array aaa if bbb can be obtained from aaa by deleting some (possibly all) elements. The product of an empty subsequence is equal to 111.

Input

The only line contains the integer nnn (2≤n≤1052≤n≤10^52≤n≤105).

Output

The first line should contain a single integer, the length of the longest subsequence.

The second line should contain the elements of the subsequence, in increasing order.

If there are multiple solutions, you can print any.

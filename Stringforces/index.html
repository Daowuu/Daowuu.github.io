Stringforces

time limit per test: 3 seconds
memory limit per test: 256 megabytes

Articles

You are given a string sss of length nnn. Each character is either one of the first kkk lowercase Latin letters or a question mark.

You are asked to replace every question mark with one of the first kkk lowercase Latin letters in such a way that the following value is maximized.

Let fif_ifi​ be the maximum length substring of string sss, which consists entirely of the iii-th Latin letter. A substring of a string is a contiguous subsequence of that string. If the iii-th letter doesn’t appear in a string, then fif_ifi​ is equal to 000.

The value of a string sss is the minimum value among fif_ifi​ for all iii from 111 to kkk.

What is the maximum value the string can have?

Input

The first line contains two integers nnn and k(1≤n≤2⋅105;1≤k≤17)k (1≤n≤2⋅105; 1≤k≤17)k(1≤n≤2⋅105;1≤k≤17) — the length of the string and the number of first Latin letters used.

The second line contains a string sss, consisting of nnn characters. Each character is either one of the first kkk lowercase Latin letters or a question mark.

Output

Print a single integer — the maximum value of the string after every question mark is replaced with one of the first kkk lowercase Latin letters.
